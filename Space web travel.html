Level 1

Use the display: flex properties to help the aliens get to their ships. The justify-content property places elements horizontally and can take the following values:

  • Flex-start – aligns all elements at the beginning
  • Flex-end – aligns all elements at the end of the container
  • Center – aligns all elements to the center
  • Space-between – elements are aligned with the same distance between them
  • Space-around – elements are aligned with the same distance around them

Level 2

Great, now we need to move the aliens not only horizontally, but also vertically. The align-items property can help with this, which can take the following values:

  • Flex-start – elements are aligned at the top edge
  • Flex-end elements are aligned at the bottom edge
  • Center – elements are aligned by cent
  • Baseline – elements are aligned on the base line of the container
  • Stretch – the elements are stretched to fill the container

Space web travel

Level 3

And here you need to put the aliens in a different order. This can be done using the flex-direction property, it sets the main direction for the axis and can take the following values:

  • Row – default value, left-to-right direction
  • Row-reverse – reverses the direction from right to left
  • Column - top-down direction
  • Column-reverse - bottom-up direction

* the flex property-the direction changes the parameters, the distribution of parameters, for which the horizontal and vertical axes are used to swap places

Space web travel

Level 4

Now the aliens need to stand in several rows. The flex-wrap property is responsible for this, which takes the following values:

  • Nowrap is the default value, all elements line up in one line
  • Wrap – allows elements to be transferred to other lines by placing them in several rows
  • Wrap-reverse – does the same as wrap, only the rows are arranged in a different order

Space web travel

Level 5

The flex-direction and flex-wrap properties are so often used together that there is a separate property for them: flex-flow. It takes the same properties as flex-direction and flex-wrap, separated by a space. For example: flex-flow: column wrap
