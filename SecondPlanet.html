Space web travel

Level 1

Help the aliens occupy the desired territory using the display: grid properties. In front of you is a grid consisting of 5 rows and 5 columns. Use the grid-column-start property to specify which column the element should be on.
The property takes integers as values: negative or positive.

Space web travel

Level 2

But what about making the element take up more than 1 column? The grid-column-end property will help with this. The property takes the same values as grid-column-start and determines where the element ends.
*instead of numbers, these properties can take the span 3 value, which determines that the element will stretch over 3 columns.

Space web travel

Level 3

In order not to write these 2 properties every time, there is a grid-column property. It takes the values grid-column-start and grid-column-end, written with a "/".
For example: grid-column: 2 / 3;

Space web travel

Level 4

And what about the arrangement of the elements line by line?
There is a grid-row-start property for this. It also accepts numeric values and determines which row the element will be placed in.

Space web travel

Level 5

Just like grid-column-end, there is a grid-row-end property that determines which row the element ends in.

Space web travel

Level 6

Now try to combine these 2 properties using grid-row.It works the same way as a grid-column.

Space web travel

Level 7

Use a common grid-column and grid-row. Get your grid area. There are 4 values indicating "/" in total: grid-the beginning of a row, grid-the beginning of a column, grid-the end of a row, grid-the end of a row.

Space web travel

Level 8

And what if the specified grid does not fit? You can resize columns using grid-template-columns. Now you need to take up half the width of the container. You can use:

  • grid-template-columns: 50% 50%
  • grid-template-columns 50% 100px
  • grid-template-columns 50% repeat(3, 50px)
  • grid-template-columns 50% 3fr 1fr

And so on...
